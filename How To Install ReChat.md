## Quest





[**With a Computer**](#with-a-computer) **(Supports Quest 1)**

[**Without a Computer**](#without-a-computer) **(Does NOT support Quest 1)**



**You do NOT need to own Rec Room on the Quest Store or join an organization to play ReChat.**





### Without a Computer

Supported headsets: Quest 2, Quest Pro, Quest 3, Quest 3S



1. Download the ReChat APK (currently unavailable.)
2. Download [AnExplorer VR File Manager](https://www.meta.com/experiences/anexplorer-vr-file-manager/6091443354290737/) from the Meta Quest Store
3. In AnExplorer, go to Downloads > rechat-quest.apk
4. Click INSTALL when the install prompt shows up (If you get a Parsing Error, troubleshoot [here](https://www.meta.com/experiences/quest/developer-post/1845596532729469/))

5\. Click OPEN when the install is done! \[=)]



To open the app in the future, filter by Unknown Sources in the App Library



### With a Computer

All PCs and Quest headsets are supported.



1. On your computer, download the ReChat APK (currently unavailable.)
2. Download Android Platform Tools for [Windows](https://dl.google.com/android/repository/platform-tools-latest-windows.zip), [macOS](https://dl.google.com/android/repository/platform-tools-latest-darwin.zip), or [Linux](https://dl.google.com/android/repository/platform-tools-latest-linux.zip).

3\. If you already have a Meta Quest developer organization **AND** you have developer mode on, skip to step 6. Otherwise, continue.

4\. In [the Meta Quest Developer website](https://developer.oculus.com/manage), create an organization. You can name it anything.

5\. In the Meta Quest mobile app, go to your headset, then headset settings, then developer mode, then turn **ON** developer mode.

6\. Plug in your headset to a USB cable on your PC.

7\. You should see a prompt - press **Always allow on this computer**. (If you don't see the prompt, plug it in while wearing the headset. If it still doesn't show up, make sure you have Developer Mode on. \[Steps 4-5])

8\. Open a Terminal in the Platform Tools folder that you downloaded in Step 2.

9\. Type **adb devices**. You should see your headset. If you don't see your headset, repeat steps 4-5. If it says Unauthorized, repeat step 7.

10\. If you see your headset and it does NOT say Unauthorized, then type **adb install /path/to/your/rechat-quest.apk**!

11\. In your headset, filter by **Unknown Sources**.

12\. Open Rec Room! (com.ChartStudiosLtd.ReChat)















# PC

Supported PCs: Windows 7 SP1 +

Do NOT run ReChat as a Non-Steam Game!!!!!!!

You do NOT need to own Rec Room on Steam to play ReChat.

You will see playtime on Spacewar; Spacewar is just the default Steam game. Yes, people abuse it for hacking tools, but you can scan it with windows defender, I assure you it is safe. :D



1. Install [ReChat Launcher](https://rechat.otiswhitaker.com/launcher/installer).
2. Open the MSI installer.
3. If it says **Windows protected your PC**, press **More info**, then **Run anyway**.
4. Let it install! You shouldn't need to interact with the installer. Just let the progress bar finish.
5. Open the Start Menu and search **ReChat Launcher**.
6. Open ReChat Launcher.
7. Download ReChat.
8. Press **Run in Screen Mode** or **Run in VR**. NEVER RUN AS A NON-STEAM GAME if you know what I mean.





# iOS

Supported Devices: iPadOS 13+

Requires a PC with macOS, Windows 7+, or Linux.

Requires an Apple ID that belongs to a user over the age of 13.



###### **Rec Room was NOT made for iPhones during this time, so iPhones are experimental! iPad is recommended!**

**(oh iPad kids...)**



1. Install [iloader](https://github.com/nab138/iloader/releases) and its dependencies.
2. Download the ReChat IPA (currently unavailable.)
3. Sign in to iloader with your Apple ID.
4. Plug in your iPhone or iPad with a Lightning or USB cable.
5. Select your device and select **Trust**. A passcode is required for this step.
6. Select **Import IPA**.
7. Select the ReChat IPA.

Done! It should install on your device like a normal app.

**You will need to reinstall ReChat every week.**
